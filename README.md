Image Center Crop
=================

Bulk center crop image with specific size.

Image Formats Support
---------------------

- PNG (IMAGETYPE_PNG) - default output format
- GIF (IMAGETYPE_GIF)
- JPEG/JPG (IMAGETYPE_JPEG)

Usage
-----

Please refer to test/index.php


Server requirement
------------------

- PHP 5.2+
- PHP GD Library
